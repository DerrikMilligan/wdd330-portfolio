Week 1 Notes

Honestly this week was mostly a review. I can re-iterate some stuff though.

  • LocalStorage: A key store that remains as a session in the users browser storage. Allows you to persist data between visits.
  • Const/let: Been using these for a long time. Const prevents you from assigning to a variable again, though for arrays/objects you can still mutate the data within.
  • Mobile First: It's easier to design for mobile and then adjust the design for larger resolutions than it is to design for large screens and attempt to scale down.
  • Users Attention: User's attention spans are pretty small and the faster and easier to use your application is the more users will use it. Google noticed they lose 20% of users per half second longer the page takes to load.
