Here's the code to make the image spin 

          
            @keyframes spinning {
              from { transform: rotate(0deg) }
              to { transform: rotate(360deg) }
            }

            .spin {
              animation-name: spinning;
              animation-duration: 3s;
              animation-iteration-count: infinite;
              /* linear | ease | ease-in | ease-out | ease-in-out */
              animation-timing-function: linear;
            }

Transitions are awesome for CSS animations and we can combine them with transforms! We tend to pick a specific property that you'd want animated when it changes, how long it'll take to apply the change, and finally if you want an easing function applied.

Animations are simlar to transitions but you specify keframe sets that define the properties at a given point and then apply those keyframes to an element in an animation.

The Canvas API is a really neat tool that allows for desiging totally custom graphics and your own rendering pipelines, if you want to do some more advanced rendering where you're in control of every pixel then this is the right tool for the job.

SVG files are raster files that can hold a set of drawing instructions to render an image or icon. Because they're just instructions for drawing, they can scale up to any size without degredation.
