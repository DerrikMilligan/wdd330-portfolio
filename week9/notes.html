One of the things we discussed this week was the Window object. In the browser you can use a window object to store globals. So you can use something like window.x = 2; to create a sharable global variable across stuff. In NodeJs we can just declare a variable in the global scope to have it global with like const x = 2;.

The window object also contains other useful stuff such as the current location which you can use to re-direct someones page window.location.href = "www.google.com";.

We also went over timing functions which are fantastic! You can use setTimeout to have something happen after a set duration.
